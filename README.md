# opensober

> Stay sober. Nothing is magic.

An opencode plugin that doesn't try to be clever.

---

## Why

Agent tools keep getting smarter, more magical, more opinionated. Prompts get injected behind
your back. Keywords flip modes. Fallback chains burn through your API quota while you sleep.

opensober goes the other way.

- **Nothing auto-runs from your prompt text.**
- **Every context injection is listed, labeled, and disable-able.**
- **No cross-model fallback.** A model that errored errors loudly.
- **One concept per job.** No category + mode + preset + skill overlap.
